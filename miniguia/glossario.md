# 📖 Glossário — AWS Cloud Practitioner

## Conceitos Fundamentais

### Cloud Computing
Entrega de recursos de TI sob demanda pela internet, com pagamento conforme o uso.

### AWS Certified Cloud Practitioner
Certificação de nível fundamental da AWS, voltada para validar entendimento geral sobre computação em nuvem.

### AWS Well-Architected Framework
Conjunto de boas práticas para projetar sistemas seguros, confiáveis, eficientes e econômicos na nuvem.

### Infraestrutura Global da AWS
Conjunto de regiões, zonas de disponibilidade, redes, hardware e data centers que sustentam os serviços da AWS.

---

## Segurança e Conformidade

### Shared Responsibility Model
Modelo que divide responsabilidades de segurança entre AWS e cliente.

### Segurança “da” Nuvem
Responsabilidade da AWS sobre a infraestrutura física, hardware, rede e data centers.

### Segurança “na” Nuvem
Responsabilidade do cliente sobre dados, acessos, configurações e permissões.

### IAM (Identity and Access Management)
Serviço usado para gerenciar identidades, permissões e acessos na AWS.

### MFA (Multi-Factor Authentication)
Camada extra de autenticação além da senha.

### Least Privilege
Princípio de conceder apenas as permissões mínimas necessárias para executar uma função.

### AWS Artifact
Portal da AWS para acesso a relatórios de conformidade e auditoria.

---

## Serviços Principais

### Amazon EC2
Serviço de servidores virtuais na nuvem.

### AWS Lambda
Serviço serverless para execução de código sob demanda.

### Amazon ECS
Serviço para orquestração e gerenciamento de contêineres.

### Amazon S3
Serviço de armazenamento de objetos.

### Amazon EBS
Serviço de armazenamento em bloco usado principalmente com instâncias EC2.

### Amazon EFS
Sistema de arquivos elástico e compartilhado para múltiplas instâncias.

### Amazon RDS
Serviço gerenciado de banco de dados relacional.

### Amazon DynamoDB
Banco de dados NoSQL totalmente gerenciado e altamente escalável.

### Amazon VPC
Rede virtual isolada dentro da AWS.

### Amazon Route 53
Serviço de DNS da AWS.

---

## Custos, Billing e Suporte

### Pay-as-you-go
Modelo de cobrança em que o cliente paga apenas pelos recursos utilizados.

### AWS Pricing Calculator
Ferramenta usada para estimar custos futuros de soluções na AWS.

### AWS Cost Explorer
Ferramenta usada para visualizar e analisar custos e uso histórico da AWS.

### AWS Budgets
Ferramenta para criação de alertas e limites de gastos.

### AWS Support
Conjunto de planos de suporte técnico oferecidos pela AWS.

# 📖 Glossário — AWS Cloud Practitioner

## AWS
Amazon Web Services, plataforma de computação em nuvem da Amazon.

## Cloud Computing
Entrega de recursos de TI pela internet, sob demanda.

## EC2
Serviço de máquinas virtuais da AWS.

## Lambda
Serviço de execução de código sem gerenciamento de servidores.

## S3
Serviço de armazenamento de objetos.

## IAM
Serviço de gerenciamento de identidades e permissões.

## MFA
Camada extra de segurança para autenticação.

## Shared Responsibility Model
Modelo que divide responsabilidades de segurança entre AWS e cliente.

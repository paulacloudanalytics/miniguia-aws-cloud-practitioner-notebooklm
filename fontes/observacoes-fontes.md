# 📝 Observações sobre as Fontes

## Critérios de seleção
As fontes foram escolhidas com base em:

- confiabilidade
- alinhamento com a certificação AWS Cloud Practitioner
- caráter oficial
- acesso gratuito
- aplicabilidade para revisão

## Por que usar fontes oficiais?
Como o objetivo do projeto é estudo para certificação, priorizar documentação e materiais oficiais ajuda a:

- evitar desinformação
- aprender a linguagem utilizada pela AWS
- focar no conteúdo realmente relevante para prova
- construir uma base sólida para estudos futuros

## Estratégia adotada
A estratégia foi usar:
1. Uma fonte central da certificação (Exam Guide)
2. Fontes complementares para segurança
3. Fontes complementares para arquitetura
4. Fontes complementares para custos e billing
